# Clasificación de Spam SMS

Proyecto de Machine Learning para clasificación de mensajes SMS como Spam o Ham utilizando técnicas básicas de NLP.

## Tecnologías utilizadas

- Python
- Scikit-learn
- TF-IDF
- Naive Bayes
- Regresión Logística
- Google Colab

## Dataset

Dataset sintético en español con 50 registros balanceados.

## Resultados

Ambos modelos alcanzaron 100% de accuracy en el conjunto de prueba.
Se incluye matriz de confusión y análisis crítico sobre el tamaño del dataset.
